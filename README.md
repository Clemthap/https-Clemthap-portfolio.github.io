👋 My Personal Website
Welcome to the repository for my personal portfolio website. This project serves as my online resume, a showcase of my work, and a central hub for my professional presence on the web.

🌐 Live Demo [(https://clemthap.github.io/https-Clemthap-portfolio.github.io/)]

📝 Description
This is a modern, responsive personal website built from scratch using pure HTML, CSS, and JavaScript. No frameworks or libraries were used, keeping the site lightweight, fast, and easy to customize.

✨ Features
🏠 About Section: A brief introduction and professional summary

💼 Portfolio/Projects: Showcase of key projects with descriptions and links

📄 Resume/CV: Embedded or downloadable resume

📱 Fully Responsive: Optimized for desktop, tablet, and mobile devices

✉️ Contact Form: Simple way for visitors to reach out

🎨 Custom Animations: Smooth scroll effects and interactive elements

⚡ Lightweight & Fast: No external dependencies or heavy libraries

🔧 Easy to Customize: Clean, well-commented code

🛠️ Built With
HTML5 - Semantic markup

CSS3 - Custom styling with Flexbox and Grid

Vanilla JavaScript - Interactive features and DOM manipulation

Font Awesome - Icons (if used)

Google Fonts - Typography
